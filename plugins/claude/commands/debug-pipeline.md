Help me debug a failed Harness pipeline execution.

Steps:
1. Ask for the pipeline name or URL (or use the most recent failed execution).
2. Use `harness_diagnose` to get failure analysis.
3. Use `harness_get` to retrieve execution details and `harness_list` for execution logs.
4. Identify the root cause — which step failed and why.
5. Suggest a fix and offer to apply it.
