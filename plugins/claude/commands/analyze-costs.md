Analyze my cloud costs using Harness Cloud Cost Management.

Steps:
1. Confirm the organization and time range for analysis.
2. Use `harness_list(resource_type="perspective")` to find cost perspectives.
3. Use `harness_get` to retrieve cost data for the relevant perspective.
4. Summarize spending by service, cluster, or cloud provider.
5. Highlight anomalies and optimization recommendations.
